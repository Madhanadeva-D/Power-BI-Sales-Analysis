## ğŸ“Š Project 1: Sales & Returns Analysis

### Features
- **Data Tables**: Order, People, Return, Order Filter Database
- **Key Metrics**: Total Sales, Total Profit, Order Quantity
- **Filters**: Category, Subcategory, Order Date, Region, Ship Date

### Reports
1. **Executive Overview** - Sales by category, profit by city, detailed matrix
2. **Customer Analysis** - Top 5 customers, YTD/QTD/MTD analysis, moving averages
3. **Product Performance** - Sales trends, product distribution, customer insights
4. **Visualization Samples** - Donut, Line, Matrix, Bar, Map charts

## ğŸ“Š Project 2: Advanced Sales Performance

### Features
- **Data Tables**: Sales, Returns, People (3-table join)
- **Key Metrics**: Total Profit, Total Quantity, Total Sales, Order Count
- **Filters**: Order Date, City, Region, Category, Sales Manager, Ship Mode, State

### Reports
1. **Sales Overview** - Manager performance, category breakdown, pie & bar charts
2. **Customer Intelligence** - Top 5 customers, manager analysis, state performance
3. **Regional Analysis** - Monthly profit trends, weekly sales, category rankings, scatter plots
4. **Advanced Charts** - Matrix, Table, Gauge, Combination charts
